data SystemTime m Source #

System time

Slots are counted from the system start.

Constructors

SystemTime 

Fields

  • systemTimeCurrent :: m RelativeTime

    Get current time (as a RelativeTime)

    For real deployment, this will take the current UTCTime and then subtract the SystemStart (see defaultSystemTime). Tests don't bother with a UTCTime and just work entirely in RelativeTime.

  • systemTimeWait :: m ()

    Wait for SystemStart

    For the real deployment, this waits for the current UTCTime to reach SystemStart. In tests this does nothing.
