data EKGForwarder req resp m a Source #

Please note that the forwarder is a client from the networking point of view: it establishes network connection with the acceptor. But after the connection is established, the forwarder becomes a server from the interaction point of view: the acceptor sends a request for new metrics, the forwarder replies to the acceptor.
