data EKGAcceptor req resp m a where Source #

Please note that the acceptor is a server from the networking point of view: the forwarder establishes network connection with the acceptor. But after the connection is established, the acceptor becomes a client from the interaction point of view: it sends a request for new metrics, the forwarder replies to the acceptor.
