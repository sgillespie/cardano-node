runRTView :: TracerEnv -> IO () Source #

RTView is a part of 'cardano-tracer' that provides an ability to monitor Cardano nodes in a real-time. The core idea is simple: RTView periodically receives some informations from the connected node(s) and displays that information on a web-page.

The web-page is built using 'threepenny-gui' library. Please note Gitub-version of this library is used, not Hackage-version!
