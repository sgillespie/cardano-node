type DatasetsIndices = TVar (Map NodeName Index) Source #

After the node is connected, we have to add a new dataset to all historical charts. The metrics received from this node will be added in these datasets. Since each dataset has its index, we need a map 'NodeName -> ix', where ix is an index of a dataset in _each_ chart. We use node's name instead of node's id, because we want to keep the same color for the same node.
